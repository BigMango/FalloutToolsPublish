# 常见问题 (FAQ)

## 安装与启动

### Q: 需要安装 .NET 运行时吗？

不需要。发布包是自包含的单文件可执行程序，无需额外安装 .NET 运行时。

### Q: 程序启动后无法自动找到 ESM 文件？

程序启动时会自动检测 Fallout 76 安装路径。若未检测到：
1. 点击 **文件 → 打开 ESM**
2. 手动选择 Fallout 76 安装目录下 `Data` 文件夹中的 `SeventySix.esm`

常见原因：游戏安装在非默认路径，或通过 Microsoft Store 安装。

### Q: 首次加载 ESM 很慢，正常吗？

正常。首次加载时程序需要：
- 解析所有记录并构建索引（约 50 万条记录）
- 构建字符串数据库（从 BA2 文件中提取）

索引和字符串数据库会被缓存，后续启动速度会明显加快。

### Q: 加载时提示找不到字符串数据库？

字符串数据库 (`strings.db`) 在首次加载时自动从游戏的 Localization BA2 文件构建。确保 ESM 所在的 `Data` 文件夹中包含 `SeventySix - Localization.ba2` 等本地化文件。

## 搜索与导航

### Q: 搜索框支持哪些搜索方式？

搜索框支持 4 种模式：

1. **关键字** — 直接输入文本，如 `Laser`，在 EditorID 和名称中模糊匹配
2. **通配符** — 使用 `*` 和 `?`，如 `Weapon*Laser`
3. **FormID 范围** — 如 `0x1000-0x2000`，搜索范围内所有记录
4. **正则表达式** — 用 `/` 包裹，如 `/^Weapon_.+Laser$/`

也可以直接输入 FormID（如 `0x0004F46A` 或 `4F46A`）按 Enter 直接跳转。

### Q: 如何使用鼠标侧键导航？

程序在全局拦截了鼠标侧键消息：
- **侧键 4（后退键）** = 导航后退（等同 `Alt + ←`）
- **侧键 5（前进键）** = 导航前进（等同 `Alt + →`）

无论焦点在哪个面板，侧键都能正常工作。

### Q: 蓝色链接是什么？如何使用？

详情面板中值列显示的蓝色带下划线文字是 FormID 引用链接。交互方式：
- **悬停** — 显示目标记录的类型、EditorID、名称
- **双击** — 跳转到目标记录
- **Ctrl + 左键单击** — 跳转到目标记录
- **右键 → 跳转到此记录** — 上下文菜单方式跳转

所有跳转都记录在导航历史中，可后退返回。

### Q: 如何快速过滤左侧树？

按 `/` 键聚焦到左侧树顶部的过滤框，输入关键字即可实时过滤。按 `Enter` 跳转到第一条匹配记录，按 `Esc` 清空过滤。

### Q: 如何过滤右侧详情面板的字段？

按 `Ctrl + G` 聚焦到右侧详情面板的字段过滤框，输入字段名或值即可过滤当前记录的子记录。按 `Esc` 清空。

## 记录分析

### Q: 如何查看一个物品的所有获取途径？

选中一条记录后，使用 **记录 → 物品产出链** 或右键菜单 → **物品产出链**。程序会追溯配方（COBJ）、容器（CONT）、NPC 掉落、等级列表（LVLI）等所有可能获取该物品的途径。

### Q: 如何对比两条记录？

1. 先查询一条记录
2. 点击 **记录 → 对比记录**
3. 在对比窗口中输入第二个 FormID
4. 左右分栏显示，差异部分高亮

### Q: 跨 ESM 对比如何使用？

需要先加载至少 2 个 ESM 文件（**文件 → 打开 ESM** 加载主文件，**文件 → 添加 ESM** 加载额外文件），然后使用 **记录 → 跨文件对比** 查看同一 FormID 在不同 ESM 中的差异。

### Q: 如何查看武器/装甲的改装选项？

选中一条 WEAP 或 ARMO 记录，使用 **记录 → 改装链**，可查看该武器/装甲所有可用的 OMOD（Object Modification）改装选项及效果。

## 导出

### Q: 如何导出记录数据？

提供多种导出方式：
- **文件 → 导出 JSON** — 导出当前记录为 JSON 文件
- **文件 → 导出截图** — 导出当前详情为 PNG 图片
- **右键 → 导出此类型 JSON/CSV** — 批量导出某类型所有记录
- **工具 → 批量导出 CSV** — 一次性导出所有类型为 CSV 文件
- **右键 → 复制完整 JSON** — 将当前记录 JSON 复制到剪贴板

### Q: 导出截图图片太大/被裁剪了？

导出截图会展开所有子记录节点并渲染完整内容。当内容过大时，高度会被限制以避免内存溢出。如果需要完整数据，建议使用 JSON 导出。

## 语言与字符串

### Q: 如何切换界面语言？

工具栏右侧有语言下拉框，选择语言后界面立即切换。支持 13 种语言。

### Q: 字符串分析器是什么？

**工具 → 字符串分析器** 可以浏览游戏内所有多语言字符串（物品名称、描述、对话等），支持搜索和导出。数据来自 `strings.db`。

## 收藏与标记

### Q: 如何使用收藏夹？

1. 在左侧树中右键记录 → **添加到收藏夹**
2. 选择或创建分组
3. 通过 **收藏夹** 菜单快速跳转到已收藏的记录

最多保存 50 条收藏，数据持久化保存，重启后自动恢复。

### Q: 颜色标记有什么用？

颜色标记用于个人标注，方便在大量记录中快速识别：
- 🔴 红色 — 重要/关注
- 🟠 橙色 — 待处理
- 🟢 绿色 — 已完成
- 🔵 蓝色 — 参考
- 🟣 紫色 — 特殊

右键记录 → **标记颜色** 即可设置，标记持久化保存。

## 性能与故障

### Q: 程序占用多少内存？

加载完整 `SeventySix.esm` 约需 1-2GB 内存，建议系统至少 4GB 可用内存。

### Q: 展开全部节点后界面卡顿？

当记录数量很多（数万条）时，展开全部节点会消耗较多渲染资源。建议：
- 仅展开需要查看的类型节点
- 使用过滤功能缩小范围
- 使用 **视图 → 折叠全部节点** 恢复

### Q: 3D 模型预览无法显示？

3D 模型预览需要 WebView2 运行时（Windows 10/11 通常已预装）。若无法显示：
- 确认系统已安装 Microsoft Edge WebView2 Runtime
- 确认 BA2 文件中包含对应的 NIF 模型文件
