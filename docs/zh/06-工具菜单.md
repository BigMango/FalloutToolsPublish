# 工具菜单 (Tools)

菜单路径: **工具(&T)**

## 字符串分析器 (&S)

- **功能**: 打开独立的字符串分析器窗口
- **数据源**: 从 `strings.db`（SQLite）中加载游戏字符串数据
- **功能详情**:
  - 按语言浏览所有游戏内字符串（物品名称、描述、对话等）
  - 支持分页浏览（每页显示固定条数）
  - 支持搜索（含通配符 `*`）
  - 可查看同一 StringID 在所有语言中的翻译
  - 支持导出当前页或全部数据
- **说明**: 若 `strings.db` 不存在，程序会在加载 ESM 时自动从 localization BA2 文件构建

## 批量导出 CSV (&B)

- **功能**: 一次性将所有记录类型导出为 CSV 文件
- **操作**:
  1. 选择导出目标文件夹
  2. 程序自动遍历所有记录类型
  3. 每种类型生成一个 CSV 文件（如 `WEAP.csv`、`ARMO.csv`）
- **CSV 内容**: FormID、EditorID、FullName
- **进度**: 状态栏显示当前导出的类型和进度

## 冲突检测 (&C)

- **前提**: 需加载至少 2 个 ESM 文件
- **功能**: 检测多个 ESM 之间的记录冲突
- **冲突类型**:
  - 同一 FormID 在不同 ESM 中存在不同数据
  - 记录覆盖检测
- **结果**: 以列表形式展示冲突记录，可双击查看详情或进行对比

## 错误检查 (&E)

- **前提**: 至少加载一个 ESM
- **功能**: 对已加载的 ESM 数据进行完整性和一致性检查
- **检查项目**:
  - 无效的 FormID 引用（引用了不存在的记录）
  - 数据结构异常
  - 字段值异常
- **结果**: 弹出窗口列出所有检查到的问题，可点击跳转到对应记录
