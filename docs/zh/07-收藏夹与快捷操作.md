# 收藏夹与快捷操作

## 收藏夹菜单

菜单路径: **收藏夹(&A)**

### 功能概述

- 将常用记录添加到收藏夹，方便快速访问
- 支持分组管理（自定义组名）
- 收藏数据持久化保存（JSON 文件），重启后自动恢复
- 最多保存 50 条收藏

### 添加收藏

1. 在左侧树中选择一条记录
2. 右键 → **添加到收藏夹 (&B)**
3. 弹出分组选择对话框，可选择已有组或输入新组名
4. 确认后记录被添加到收藏夹

### 管理收藏

- **移动到收藏组 (&M)**: 右键已收藏的记录，将其移动到其他分组
- **从收藏夹移除 (&D)**: 右键移除该记录的收藏
- **清空收藏夹**: 通过收藏夹菜单底部的「清空收藏夹」选项

### 使用收藏

- 点击收藏夹菜单，展开分组列表
- 单击任意收藏项即可跳转到该记录
- 每个收藏项显示 `[FormID] EditorID (Type)` 格式

## 颜色标记

在左侧树中右键记录 → **标记颜色(&T)**:

- 🔴 **红色** — 重要/关注
- 🟠 **橙色** — 待处理
- 🟢 **绿色** — 已完成
- 🔵 **蓝色** — 参考
- 🟣 **紫色** — 特殊
- **清除标记** — 移除颜色

标记的颜色会在左侧树节点上直接显示，持久化保存。

## 左侧树右键菜单

在左侧记录树上右键，可用操作:

| 菜单项 | 快捷键 | 说明 |
|-------|--------|------|
| 查看引用者 (&R) | — | 查找引用当前记录的所有记录 |
| 物品产出链 (&P) | — | 分析物品的产出来源 |
| 复制 FormID (&I) | — | 复制格式为 `0x00XXXXXX` 的 FormID |
| 复制 EditorID (&E) | — | 复制编辑器 ID |
| 添加到收藏夹 (&B) | — | 添加到收藏夹（含分组） |
| 移动到收藏组 (&M) | — | 移动到其他收藏组 |
| 从收藏夹移除 (&D) | — | 从收藏夹中删除 |
| 标记颜色 (&T) | — | 设置/清除颜色标记 |
| 导出此类型 JSON (&J) | — | 导出当前类型所有记录为 JSON |
| 导出此类型 CSV (&C) | — | 导出当前类型所有记录为 CSV |
| 查看地图 (&W) | — | Worldspace 节点专用，查看 CELL 地图 |

## 右侧详情面板右键菜单

在右侧详情树上右键，可用操作:

| 菜单项 | 快捷键 | 说明 |
|-------|--------|------|
| 复制 FormID (&I) | — | 复制当前记录的 FormID |
| 跳转到此记录 (&G) | — | 跳转到当前行引用的 FormID |
| 复制字段值 (&V) | — | 复制当前单元格的值 |
| 复制当前节点 JSON (&N) | — | 将当前节点及子节点导出为 JSON |
| 复制完整 JSON (&F) | — | 将整条记录导出为 JSON |

## 键盘快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + F` | 搜索记录（聚焦搜索框） |
| `Ctrl + Shift + F` | 高级搜索 |
| `Ctrl + G` | 聚焦详情字段过滤框 |
| `Alt + ←` | 导航后退 |
| `Alt + →` | 导航前进 |
| `F3` | 循环跳转最近浏览记录 |
| `/` | 聚焦左侧树过滤框 |
| `Esc` | 清空当前聚焦的过滤框 |
| `Enter` | 在搜索框中执行搜索 |
| `Enter` | 在过滤框中跳转到第一条匹配记录 |

## 鼠标操作

| 操作 | 功能 |
|------|------|
| **左键单击** 左侧树节点 | 加载记录详情 |
| **双击** 详情面板蓝色链接 | 跳转到引用的 FormID |
| **Ctrl + 左键单击** 蓝色链接 | 跳转到引用的 FormID |
| **鼠标侧键 4** (XButton1) | 导航后退 |
| **鼠标侧键 5** (XButton2) | 导航前进 |
| **悬停** 蓝色链接 | 显示目标记录的 EditorID/类型/名称提示 |
