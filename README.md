# Fallout76 Data Tools

A tool for parsing and browsing Fallout 76 ESM files, similar to xEdit.

## Features
- ESM Browser with FormID / EditorID / Name search
- Multi-language support (13 languages)
- 3D Model Preview (WebView2)
- Texture Preview
- Record comparison & cross-ESM compare
- Field-level search
- Navigation history & bookmarks
- Cell map viewer
- JSON / CSV / Screenshot export

## Download
See [Releases](https://github.com/BigMango/FalloutToolsPublish/releases) for the latest build.

## System Requirements
- Windows 10/11 x64
- No .NET runtime required (self-contained)
